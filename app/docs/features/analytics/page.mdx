# Analytics

Track your store's performance with the analytics dashboard.

> **Pro Feature:** Analytics is available on Pro and Hosted tiers. [Compare tiers](/docs/tiers)

## Dashboard Overview

Access analytics at **Admin > Analytics**.

The dashboard shows:
- Revenue over time
- Order count
- Top-selling products
- Order status breakdown

## Time Periods

View data for different ranges:
- **7 Days** - Last week
- **30 Days** - Last month
- **90 Days** - Last quarter

Select the period in the top-right corner.

## Revenue Chart

The line chart shows daily revenue:
- Y-axis: Dollar amount
- X-axis: Date
- Hover for exact values

Use this to:
- Spot trends
- Track promotion impact
- Compare periods

## Orders Chart

The bar chart shows orders per day:
- Height = number of orders
- Color-coded by status

Useful for:
- Understanding demand patterns
- Planning fulfillment resources
- Identifying busy days

## Top Products

See your best sellers:
- Product name
- Units sold
- Revenue generated

Insights:
- Focus marketing on top performers
- Ensure adequate inventory
- Consider bundles with popular items

## Order Status Breakdown

Pie chart showing order distribution:
- Pending
- Processing
- Shipped
- Delivered
- Cancelled

A healthy store typically shows:
- Most orders as Delivered
- Few cancellations
- Quick turnover from Pending to Shipped

## Export Data

Download your analytics as CSV:

1. Click **Export CSV**
2. Select date range
3. File downloads with:
   - Date
   - Order count
   - Revenue
   - Product breakdown

Use exports for:
- Accounting
- Tax preparation
- External analysis

## Key Metrics

### Revenue
Total sales after discounts, before Stripe fees.

### Orders
Number of completed purchases.

### Average Order Value
Revenue รท Orders. Higher is better.

### Conversion Rate
Not tracked in current version. Use Google Analytics for traffic metrics.

## Tips

### Regular Review
Check analytics weekly to catch trends early.

### Compare Periods
Look at month-over-month changes, not just absolute numbers.

### Act on Insights
- Low revenue? Run a promotion
- Popular product? Feature it prominently
- Many cancellations? Improve product descriptions

### External Tools
For deeper analytics, consider:
- Google Analytics (traffic, sources)
- Stripe Dashboard (payment analytics)
- Your email provider (campaign performance)
