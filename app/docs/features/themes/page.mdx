# Themes

Customize your store's appearance with theme presets and brand colors.

## Theme Presets

GoSovereign includes professionally designed theme presets:

| Theme | Style | Tier |
|-------|-------|------|
| **Default** | Clean, modern, neutral | All |
| **Minimal** | Simple, lots of whitespace | All |
| **Bold** | Strong colors, high contrast | All |
| **Warm** | Earthy tones, inviting | Pro |
| **Cool** | Blues and grays, professional | Pro |
| **Luxury** | Dark mode, premium feel | Pro |

## Changing Your Theme

1. Go to **Admin > Settings**
2. Click the **Appearance** tab
3. Browse theme presets
4. Click a theme to preview
5. Click **Save** to apply

Changes take effect immediately.

## Theme Components

Each theme controls:

### Colors
- **Brand Color** - Buttons, links, accents
- **Background** - Page background
- **Text** - Primary text color
- **Borders** - Dividers and outlines

### Typography
- Font weights and sizes are optimized per theme
- All themes use system fonts for fast loading

### Components
- Button styles
- Card appearances
- Navigation styling
- Footer design

## Brand Color

Your brand color is used for:
- Primary buttons
- Links
- Selection highlights
- Active states
- Loading indicators

### Choosing a Brand Color

1. During setup, pick from the color palette
2. Or enter a custom hex code
3. Preview how it looks across your store

**Tips:**
- Choose a color with good contrast against white
- Avoid colors too close to red (can look like errors)
- Test on both desktop and mobile

## Starter vs Pro Themes

### Starter Tier
Access to 3 basic themes:
- Default
- Minimal
- Bold

### Pro Tier
Access to all 6 themes including premium options:
- Warm
- Cool
- Luxury

Upgrade to Pro to unlock all themes: [Pricing & Tiers](/docs/tiers)

## Dark Mode

The **Luxury** theme (Pro) features a dark color scheme:
- Dark background
- Light text
- Gold accents

Ideal for:
- Premium/luxury products
- Night-focused audiences
- Standing out from competitors

## Mobile Appearance

All themes are fully responsive:
- Navigation adapts to mobile menu
- Product grids stack vertically
- Touch-friendly buttons
- Optimized image sizes

## Tips

### Consistency
Stick with one theme. Changing frequently can confuse customers.

### Test on Mobile
Most shoppers browse on phones. Check your store on mobile devices.

### Brand Alignment
Choose a theme that matches your brand personality:
- Professional services → Minimal or Cool
- Handmade/artisan → Warm
- Tech products → Default or Bold
- Luxury goods → Luxury
